Como criar o Bot do Telegram?

  1. Fale com o @BotFather no Telegram e envie /newbot para criar um bot.
  2. Copie o Token de API fornecido pelo BotFather e cole no campo acima.
  3. Crie um canal público ou grupo no Telegram e adicione seu bot recém-criado como Administrador (com permissão de postar mensagens).
  4. Obtenha o ID do seu canal (se for público, use @nomedocanal; se for privado, você pode encaminhar uma mensagem dele para o bot @RawDataBot para descobrir o ID).
